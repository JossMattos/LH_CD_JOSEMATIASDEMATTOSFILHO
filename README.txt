Passos para execução do projeto:

-> Instale o Jupyter Notebook no seu desktop;

-> Ao finalizar a instalação, a inicialização do Jupyter deve ocorrer no seu navegador web, na guia de nome "Home";

-> Faça o upload dos arquivos da pasta LH_CD_JOSEMATIASDEMATTOSFILHO para o diretório principal. Todos os arquivos devem estar no mesmo diretório;

-> É muito importante a base de dados (arquivo 'desafio_indicium_imdb.csv') estar no mesmo diretório que 'imdb_Rating.ipynb', já que a base de dados será endereçada para execução do arquivo 'imdb_Rating.ipynb'.

-> Localize e execute o arquivo 'imdb_Rating.ipynb';

-> Quando o algoritmo abrir e você visualizar o código, execute-o sequencialmente, clicando no símbolo triangular na parte
superior esquerda da sua tela. Analise e interaja com os blocos de código;

-> No penúltimo bloco de código '#Inserção de novos dados', você pode alterar os dados e obter a previsão desejada. 

-> A cada execução, o arquivo 'modelo_treinado.pkl' armazenará os dados de treinamento de 'model' durante a execução de 'imdb_Rating.ipynb'. Você poderá localizar este arquivo no mesmo diretório que todos os arquivos já citados neste documento.
